# PDF 论文提取规则

## 一、页面编号

PDF 页面序号是 PDF 阅读器中的页面顺序。论文印刷页码是论文页面底部或页眉处显示的页码。两者可能不同。

输出页码时必须说明使用的是哪一种。如果能同时识别两者，应同时列出。

## 二、扫描版 PDF

扫描版 PDF 通常无法直接提取文本。遇到扫描版时：

- 说明需要 OCR。
- 不得假装已经读取正文。
- 可以基于可见页面截图进行有限判断，但必须说明限制。

## 三、目录和章节识别

优先根据目录识别章节。若目录缺失，则根据正文标题识别章节。

常见章节包括：

- 摘要。
- Abstract。
- 目录。
- 绪论。
- 需求分析。
- 系统设计。
- 系统实现。
- 系统测试。
- 结论与展望。
- 参考文献。
- 致谢。
- 附录。

## 四、图表识别

应记录：

- 图表编号。
- 图表标题。
- 所在页码。
- 所属章节。
- 正文是否引用。

如果不能识别图表具体内容，只能说“已定位图表”，不能判断图表是否合理。

## 五、可审查范围说明

输出前应说明本次已经成功读取的范围。例如：

- 已读取全文文本。
- 已读取目录和正文，但图表内容不可识别。
- 仅能读取部分页面。
- 需要用户补充截图或 OCR 文本。
